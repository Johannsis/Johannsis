## 12) Profile About Card Script

<a href="#">
  <picture>
    <source media="(prefers-color-scheme: dark)" srcset="https://raw.githubusercontent.com/Johannsis/Johannsis/main/assets/dark_mode.svg">
    <img alt="Johannsis's GitHub Profile README" src="https://raw.githubusercontent.com/Johannsis/Johannsis/main/assets/light_mode.svg">
  </picture>
</a>

The custom SVG stats card generated by `scripts/about.ts` was inspired by the
profile README in:
[https://github.com/Andrew6rant/Andrew6rant](https://github.com/Andrew6rant/Andrew6rant)

This repository reimplements that idea in TypeScript and adapts the layout,
rendering, and GitHub data collection for this profile README's About section.

The ASCII art in `assets/ascii-art.txt` was created from a profile picture with
a transparent background using the converter at
[https://www.asciiart.eu/image-to-ascii](https://www.asciiart.eu/image-to-ascii).

Reference files in this repository:

```text
scripts/about.ts
assets/ascii-art.txt
assets/dark_mode.svg
assets/light_mode.svg
.github/workflows/update-about.yml
```

Example (HTML):

```html
<a href="#">
  <picture>
    <source
      media="(prefers-color-scheme: dark)"
      srcset="
        https://raw.githubusercontent.com/Johannsis/Johannsis/main/assets/dark_mode.svg
      "
    />
    <img
      alt="Johannsis's GitHub Profile README"
      src="https://raw.githubusercontent.com/Johannsis/Johannsis/main/assets/light_mode.svg"
    />
  </picture>
</a>
```
